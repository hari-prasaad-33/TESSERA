VIDEO FILES - TESSERA WEBSITE
============================

Place your video files in this folder with these exact names:

1. founders-video.mp4
   → Founder's message (shown on home page, after "The Human Element" section)

2. tessera-one-demo.mp4
   → Plugin demo (shown on Discover Plugin page)

Supported formats: mp4 (recommended for compatibility), webm

If you use different filenames, update the paths in:
  - src/components/FounderVideoSection.jsx  (FOUNDER_VIDEO_SRC)
  - src/components/DiscoverPage.jsx         (DEMO_VIDEO_SRC)
